🔌 Input Power

  • AC mains input of 230V at 50Hz
  • Connected to a step-down transformer (230V to secondary, 1.5A)

🔃 Rectification Stage

Full-wave rectifier using 4x 6A4 diodes in bridge config.

🔧 Smoothing Stage

470μF capacitor (35V) filters the DC output.

🔋 Voltage Regulation Stage

Uses 7815 and 7805 ICs for +15V and +5V outputs.

🌀 Cooling Mechanism

Heat sink + cooling fan on voltage regulators.

⚡ Output Stage

300W 10A 30V Buck Converter controls final output.

🎚️ Control and Switching System

  • 10kΩ Potentiometers for voltage/current
  • Black switch: main power control
  • Red switch: electroplating ON/OFF

Try to control wirelessly via LAN using Raspberry Pi

📟 Display System

Digital Voltmeter + Ammeter to show output. Measures using buck converter + shunt resistor.

💡 LED Indicators

  • Red LED: Process indicator
  • Other LEDs: Power status & buck converter ON

⚠️ Circuit Precautions

  • Limit current to 5-8A
  • Use proper insulation and cooling
  • Good soldering ensures durability
